# Summary

[![View on GitHub](https://img.shields.io/github/last-commit/mkellsy/reprap-delta.svg?colorB=%23999999&label=updated&logo=github&logoColor=%2364b5f6&style=popout-square)](https://github.com/mkellsy/reprap-delta)

This is a full body, one piece carriage for the HE3D K280, and should also work for the HE3D K200. Plus this should work for any delta/kossel that uses 2020 extrusions.

This also has a belt tensioner, inspired by Nanocube's design. I also added belt lock plates, which should keep the belts nice and snug.

## Assembly
To assemble you will need;

- A M3 tap (Optional).
- 2 x M3 x 6mm socket head cap screws.
- 6 x M3 x 20mm socket head cap screws.
- 4 x M5 x 30mm button head screws. (Reused)
- 6 x M3 Nuts.
- 4 x M5 Nuts. (Reused)
- 2 x 7mm Spacers. (Reused)
- 2 x Offset Spacers. (Reused)
- 4 x Washers. (Reused)
- 2 x Thick Copper Washers. (Reused)
- 4 x Wheels. (Reused)
